# Stage 7: 测试评审清单

## 覆盖率

- [ ] 是否覆盖所有功能需求
- [ ] 是否覆盖边界条件
- [ ] 是否覆盖异常路径
- [ ] 是否覆盖集成场景

## 测试类型

- [ ] 单元测试是否覆盖核心逻辑
- [ ] 集成测试是否覆盖模块间交互
- [ ] 端到端测试是否覆盖关键用户路径
- [ ] 是否有性能测试（如需要）

## 可执行性

- [ ] 测试命令是否可运行
- [ ] 测试环境是否可复现
- [ ] 测试数据是否可准备
- [ ] 测试是否稳定（非 flaky）

## 验证闭合

- [ ] 测试结果是否与预期一致
- [ ] 失败的测试是否有明确的修复计划
- [ ] 是否有未测试但有风险的代码路径
