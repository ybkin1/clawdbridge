# Command Catalog

> Claude Code 内置工具命令速查，用于 task-directory-tree 的 `references/command-catalog.md`。

## 文件操作

| 命令 | 用途 |
|------|------|
| `Read` | 读取文件内容 |
| `Write` | 创建/覆盖文件 |
| `Edit` | 精确替换文件内容 |

## 搜索操作

| 命令 | 用途 |
|------|------|
| `Grep` | 按内容搜索 |
| `Glob` | 按文件名模式搜索 |

## 执行操作

| 命令 | 用途 |
|------|------|
| `Bash` | 执行 shell 命令 |

## 其他操作

| 命令 | 用途 |
|------|------|
| `Agent` | 派发子 Agent |
| `WebSearch` | 互联网搜索 |
| `WebFetch` | 获取网页内容 |
| `TodoWrite` | 任务跟踪 |
| `AskUserQuestion` | 向用户提问 |
